# Rodar amanha

Este projeto nao precisa instalar dependencias nesta versao.

## Jeito mais simples

1. Abra a pasta `C:\WINDOWS\system32\pokemon-tcg-erp`.
2. Dê dois cliques em `start-erp.bat`.
3. Abra `http://localhost:3000`.

## Pelo PowerShell

```powershell
cd C:\WINDOWS\system32\pokemon-tcg-erp
.\start-erp.ps1
```
